# Project plans

Planning files live here so the repository root stays focused on package,
examples, docs, and tests.

- `plan_research_grade_performance_differentiability.md`: current umbrella plan
  and status log for performance, differentiability, validation, and refactor
  work.
- `plan_differentiability.md`: compact historical differentiability/refactor
  notes.
- `plan_freeb.md`: historical free-boundary evidence summary.
- `plan.md`: older project-wide planning notes.
- `discrete_adjoint_2506_plan.md`: historical notes for the 2025/2026
  spectral-adjoint work.

Update only the umbrella plan for new active work unless a historical note is
being corrected.
