Mirror Geometries
=================

.. warning::

   Mirror geometry support is experimental. The current implementation contains
   fixed-boundary scalar-pressure mirror geometry, fields, residuals,
   manufactured-solution checks, mirror-native ``mout`` output, plotting,
   cap-to-cap field-line pitch diagnostics, and optimizer prototypes.
   Free-boundary mirrors, anisotropic pressure, stellarator-mirror hybrid
   boundaries, and the final differentiable implicit-solve API remain planned
   work.
