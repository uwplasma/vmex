# Postprocessing Tests

Tests in this folder cover analysis and visualization workflows that sit on top
of VMEC equilibria: Boozer inputs/CLI behavior, plotting helpers, field-line
diagnostics, pressure/current profile examples, bootstrap-current examples, and
Glasser/Mercier diagnostic plots.

Keep WOUT file-format tests in `tests/io/wout/`; keep solver convergence in
`tests/solvers/`; keep VMEC2000/VMEC++ agreement in `tests/parity/`.
