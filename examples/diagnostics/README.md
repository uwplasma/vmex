# Diagnostic Examples

This folder contains small user-facing scripts for inspecting or plotting VMEC
outputs. They are examples, not release gates.

Use these scripts to learn how to:

- load a WOUT file and print common profiles,
- compute or plot derived quantities such as `DMerc` and `D_R`,
- make simple diagnostic figures from saved equilibria.

Generated plots and WOUT files should be written to ignored `results/`,
`outputs/`, or a user-selected scratch directory.
