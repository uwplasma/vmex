# Single-grid benchmark inputs

These are fixed-boundary single-grid VMEC input decks used for runtime,
VMEC2000/VMEC++ parity, and physics-gate validation.

Large reference files such as `wout_*_reference.nc` and `mgrid_*.nc` are fetched
on demand and ignored by git.  The checked-in files are intentionally limited to
small VMEC input decks and compact fixtures.
